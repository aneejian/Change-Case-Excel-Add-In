Change Case Excel Add-In

Change Case Excel Add-In is a powerful Excel VSTO Add-In that can change case of selected cell or range of cells. With easy to use Keyboard shortcuts, perform case conversion in style.

Features

Capability to change case of selected cell or range of cells

  • UPPER CASE
  • lower case
  • Sentence Case
  • Capitalize Each Word
  • tOGGLE cASE
  • aLtErNaTiNg CaSe

Other features

  • Customizable keyboard shortcuts
  • Multiple ways to access
    •  Home Tab
    •  Change Case Tab
    •  Cell context menu

  •  for ‘tOGGLE cASE’
  •  for ‘aLtErNaTiNg CaSe’

Customize Shortcuts

Use the settings menu to customize Keyboard Shortcuts or disable it completely.

  • Open an Excel file
  • Right click on a cell
  • Roll over the 'Change Case' option in the menu
  • Select the option 'Settings'
  • In the settings menu you can change the default shortcuts
  • From the context menu or the settings window you can enable or disable shortcuts
